# Forensik Teknologi Informasi

Repositori ini berisi materi, studi kasus, dan tools yang digunakan dalam analisis forensik digital.

## 🔍 Konten

- 📂 `kasus/`: berisi file kasus dan bukti digital (disk image, log, dll)
- 🛠️ `tools/`: skrip otomatisasi dan setup tools
- 📄 `laporan/`: laporan hasil investigasi

## 🧰 Tools yang Digunakan

- Autopsy
- FTK Imager
- Volatility
- Wireshark
